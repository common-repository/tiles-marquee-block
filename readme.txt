Add a marquee banner block to your Gutenberg built website with the Tiles Marquee block.

== Description ==

A quick and simple way to add a creative marquee banner to your website.

== Screenshots ==
1. Add an animated marquee to your website.

== Installation ==
1. Upload the `tiles-marquee-block` folder to your `/wp-content/plugins/` directory or alternatively upload the tiles-marquee-block.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your local computer.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Search for the block within the Gutenberg block editor.

== Frequently Asked Questions ==

= Can this block be used on my theme? =
Yes! You can use this block on any Gutenberg block-based, full site editing, or universal WordPress theme. 

= What's Tiles? =
Tiles is a new project we're ramping up, geared towards building a full-fledged design framework with Gutenberg block patterns that can be composed to build any design, directly within the Gutenberg block editor.
